# websocket and socketio chat room
两个都是多人聊天室，具体功能有：
用户进来时聊天室会广播，同样的用户起开的时候也会向全体广播;
用户可以在聊天室发送消息；
右下角有用户的id；
具体如下：
![示例](https://raw.githubusercontent.com/nofish/Markdown-Photos/master/websocket-and-socketio-chat-room/1.png)
使用方法：
>在命令行工具下进入文件夹；
安装npm包 npm install
输入 npm start
